# SentimentTools

Analisis sentimen konten web, jalan 100% lokal.

## Quick Start
```bash
cd sentiment-tools
pip install -r requirements.txt
python scripts/download-models.py
cp .env.example .env
python -m src.main
```

## Docker
```bash
docker compose up --build
```

## Requirement
- Python 3.11+
- RAM 4GB+
- Disk 2GB+
